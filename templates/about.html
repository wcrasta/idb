Tools

Bootstrap:

Bootstrap is an HTML/CSS/JS framework used to aid in the design of our website. It provided a library of useful and fluid design tools, such as the splash page's carousel, navbar and more.

React:

React is a Javascript framework used to aid in the functionality of our website. We used it to link things such as the search bar to our database.

Flask:

Flask is a Python framework used to help implement our backend and serve our website.

SQLAlchemy:

SQLAlchemy is a Python-SQL framework used to build and query our database on the backend.

Apiary:

Apiary is a tool used to build/manage/document our API. We then use this to allow others to scrape our site.

Namecheap:

Namecheap is a service that we used to claim our domain for our website.

Amazon Web Services (AWS):

AWS is a website hosting / management service, served by Amazon, that we used to deploy our website with.

Slack:

Slack is a team communication application that we used to discuss design decisions, scheduling and much more. We also integrated our GitHub repo directly to our main chat channel so that we could all receive real-time updates on the project.

GitHub:

GitHub is the version control site we used to store our project files and documentation, with the help of git.
